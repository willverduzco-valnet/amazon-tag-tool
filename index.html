Amazon Tag Tool

Notes
  • This tool is intended for use with any Amazon.com (US) URL. It intentionally does not work with Amazon.ca or any other regional Amazon store to prevent region-locked deals issues.
  • The Deals Newsletter Builder tab automatically affiliates all Amazon URLs. There is no need to use this tab when building a newsletter.
  • The URL is stripped down to just the ASIN (product ID). All tracking parameters, referral codes, and slug text from the original link are removed.
  • AP uses a non-standard newsletter tag (ap-newsletter04-20). All other brand/purpose combinations follow the standard naming pattern.
